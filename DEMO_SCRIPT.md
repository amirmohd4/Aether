## 🎨 Visual Style Guide

### **Color Scheme:**
- Primary: Blue (#2563EB)
- Success: Green (#10B981)
- Warning: Orange (#F59E0B)
- Danger: Red (#EF4444)
- Background: Light Gray (#F9FAFB)

### **Transitions:**
- Smooth fades between scenes (0.5s)
- Quick cuts for action sequences
- Zoom-in on important numbers/badges
- Highlight cursor for emphasis

### **Music:**
- Modern, upbeat corporate music
- Low volume, don't overpower voiceover
- Tempo: 120-130 BPM

### **Text Overlays:**
- Font: Inter or Poppins (sans-serif)
- Size: Large for titles, medium for stats
- Animation: Slide-in from bottom

## 🎬 Production Notes

**Recording Setup:**
- Screen resolution: 1920x1080
- Browser: Chrome (clean profile, no extensions visible)
- Cursor: Large, visible cursor
- Zoom: 110-120% for readability

**Post-Production:**
- Add subtle motion blur
- Color grade for consistency
- Add sound effects for clicks/transitions
- Compress for web (H.264, 1080p, 5-8 Mbps)

**Deliverables:**
- MP4 (1080p) - YouTube, website
- MP4 (720p) - Twitter, LinkedIn
- GIF (480p) - Email, social media snippets
